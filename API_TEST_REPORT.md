### 测试用例3: 英文对话
```bash
curl -X POST http://localhost:8080/api/v1/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "List 3 key components of GDPR compliance"}'
```
**结果**: ✅ 通过
- 模型能够理解问题并生成相关回答
- 提到了DPO、数据主体权利等GDPR核心要素

### 测试用例4: 中文对话
```bash
curl -X POST http://localhost:8080/api/v1/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "请简要介绍数据合规的重要性"}'
```
**结果**: ✅ 通过
- 模型能够理解中文并用中文回复
- 内容涵盖法律风险、声誉维护、业务可持续性等方面
